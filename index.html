Jostle link

A recreation of an animation technique by PilotRedSun. The way it works is the program selects every fourth pixel in a 1230x720 image, creating a 640x360 image. 3 other unique 360p images can be created as well, which all go into a GIF, which in this case cycles through the frames at I believe I set to be 10 fps.

Panama link

A simple message encryptor designed for encrypting and decrypting 7 bit signed ascii. It uses a method which I now recognize to be a Vigenere cipher. This was a project I did with a friend and is useless for actual encryption, although I think it has potential if the key is long enough.
